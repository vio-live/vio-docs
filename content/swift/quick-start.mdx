# Quick Start

Connect the Vio SDK to your streaming app in under 10 minutes.

## 1. Add the Package

In Xcode: **File â†’ Add Package Dependencies...**

```
https://github.com/angelosv/VioSwiftSDK.git
```

Select version `0.1.0-alpha` and add `VioCore`, `VioEngagementUI`, `VioCastingUI` to your target.

## 2. Create `vio-config.json`

Add a `vio-config.json` file to your Xcode project (**Copy Bundle Resources**):

```json
{
  "apiKey": "YOUR_VIO_API_KEY",
  "environment": "production",
  "campaigns": {
    "webSocketBaseURL": "wss://api.vio.live",
    "restAPIBaseURL": "https://api.vio.live",
    "campaignApiKey": "YOUR_CAMPAIGN_API_KEY",
    "autoDiscover": true
  },
  "theme": {
    "mode": "automatic"
  }
}
```

> Your API key and campaign API key are available in the **Vio Dashboard â†’ Apps**.

## 3. Load Configuration at Startup

```swift
import VioCore

@main
struct MyApp: App {
    init() {
        // Loads vio-config.json from the app bundle automatically
        ConfigurationLoader.loadConfiguration(userCountryCode: "NO")
    }

    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

## 4. Connect to a Broadcast

Call `setBroadcastContext` with the `contentId` from your streaming platform. This triggers campaign discovery and WebSocket connection:

```swift
import VioCore

// When the user opens a stream
let context = BroadcastContext(
    broadcastId: "real-madrid-vs-barcelona-2025-01-24",  // matches externalId in Dashboard
    broadcastName: "Real Madrid vs Barcelona"
)

await CampaignManager.shared.setBroadcastContext(context)
```

From this point, the SDK:
1. Calls the backend to find the active campaign for this broadcast
2. Opens a WebSocket connection
3. Polls and contests are pushed in real-time as you configured them in the Dashboard

## 5. Show the Engagement Overlay

Use `VioEngagementOverlay` (from `VioCastingUI`) to display polls and contests on top of your video player:

```swift
import SwiftUI
import ViaCastingUI

struct StreamView: View {
    var body: some View {
        ZStack {
            // Your video player
            VideoPlayerView()

            // Vio engagement overlay â€” handles polls, contests, sponsor badge automatically
            VioEngagementOverlay()
        }
        .task {
            let context = BroadcastContext(
                broadcastId: "your-broadcast-external-id"
            )
            await CampaignManager.shared.setBroadcastContext(context)
        }
    }
}
```

## 6. Disconnect When Stream Ends

```swift
// When the user exits the stream
CampaignManager.shared.disconnect()
```

## What Happens Automatically

Once `setBroadcastContext` is called:
