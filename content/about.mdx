# About Vio

Vio is a **live engagement SDK** for sports streaming platforms. It adds interactive polls, contests, and shoppable products to live streams — synchronized with the broadcast in real-time.

## What Vio Does

- **Polls** — Timed prediction questions that appear at configured match minutes
- **Contests** — Sponsored competitions with prizes
- **Shoppable Products** — Sponsor product carousels and spotlights driven by the Dashboard
- **WebSocket-driven** — All events are pushed in real-time, no polling required

## Architecture

The **Dashboard** is the control plane — broadcasters configure campaigns, broadcasts, and engagement elements. The **SDK** (Swift + Kotlin) receives everything at runtime via API and WebSocket.

No code changes are needed to change what products appear, when polls fire, or which sponsors are active during a match.
